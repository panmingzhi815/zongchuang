# vux与apicloud 项目模板

##### 现在用apicloud开发的app，基本每个界面都是一个独立的html文件，虽然上手比较快，但复用的概率太小。用vux虽然上手慢一点，但熟悉后，代码量极少。

#### 涉及功能网站
###### apicloud官网：http://www.apicloud.com
###### vux官网：https://vux.li/

#### 基本的运行方式就不多讲了
###### npm install
###### npm run dev
